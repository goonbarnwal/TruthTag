TruthTag API Documentation

Overview

TruthTag API allows developers to integrate job scam detection capabilities into their applications. The API analyzes job postings and returns detailed risk assessments.

POST /api/v1/analyze-job

Analyze Job Posting

Submit job details for analysis and receive a comprehensive risk assessment.

Request Body
{ "company_name": "string", "hr_email": "string", "job_description": "string", "job_url": "string (optional)" }
Response
{ "risk_score": 65, "risk_level": "suspicious", "confidence": 0.92, "factors": [ { "type": "email", "severity": "high", "description": "Uses free email domain", "recommendation": "Verify company contact" } ], "analysis_time": "2023-12-01T10:30:00Z" }
